# Data Jobs Dashboard w/Power Bi

![First page](/images/First%20page.png)


## Introduction

This dashboard helps **job seekers, career changers and professionals** find clear information about the data job market, which is usually scattered and hard to track. Using a dataset of 2024 data science job postings, this project combines job titles, salaries and locations into one easy to use interface. It provides a central place to quickly understand current market trends and compensation.

## Skills Showcased

Data Transformation with Power Query
<br>Implicit Measures
<br>Core Charts
<br>Geospatial Analysis
<br>KPI Indicators & Tables
<br>Dashboard Design
<br>Interactive Reporting(Slicers, Buttons & Bookmarks, Drill-Through)

## Dashboard Overview

### Page 1 : High Level Market View

![First page](/images/First%20page.png)

This dashboard serves as a central control center for analyzing the data job market.It showcases key KPIs including total job counts, median salaries and top job title to provide a clear, at a glance understanding of market trends and compensation structures.

![Second page](/images/Second%20Page.png)

This deep dive page lets you drill through from the main dashboard to see specific details for any single job title. It shows clear details about salary ranges, remote work statistics, top hiring platforms and a global map of job locations.

## Conclusion

This project provides a powerful, user friendly tool for navigating the often complex data job market. By combining high level market trends with detailed role analysis, the dashboard helps users make informed career decisions based on solid data. Whether you are searching for your next role, changing careers or analyzing global compensation, this interface delivers the clear insights needed to succeed.
